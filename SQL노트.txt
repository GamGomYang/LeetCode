2. 인자 선언 DECLARE 
    예시 : DECLARE M INT ; 
           SET M = N-1 ;
    이렇게 해야함 
    but DECLARE 해보니까 그냥 SET하는거보다 시간이 오래걸림   

3. 윈도우 함수 LAG - 이전을 보세용 함수
       사용법 : LAG(num) OVER (ORDER BY id)
       - 지금 num의 1칸전을 봐라
       - LAG(num , 2) : 2칸전을 봐라

       * 반대되는 윈도우 함수 LAED()
       - 다음 행 값을 가져온다

4. 상관 서브쿼리 - 안쪽쿼리가 바깥쪽 쿼리 참조

       - e2 작동할때 마다 바깥 e 테이블 참조

10. SQL에서의 if else - CASE WHEN THEN END문
    SELECT
    id,
       CASE
        WHEN p_id IS NULL THEN 'Root'
        WHEN id IN (SELECT p_id FROM Tree) THEN 'Inner'
        ELSE 'Leaf'
       END AS type
     FROM Tree;
